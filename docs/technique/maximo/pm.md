# PM

## create 工單

```
MAXIMO dashborad:
  要注意填表的組織filter
1. (option) New a type to organization
2. Setup “規劃”-> “路徑”(Route). 這是設定create 子母工單的collection
3. Setup “規劃”-> ”工作計畫”. 這是設定工單的task
4. Setup “PM” -> 選擇“路徑”跟“作業計畫” => 子母工單
            -> 選擇“資產”跟”作業計畫” => 只有母單(本身有工作計畫)
            => “設定工作計畫順序” 還有frequence
```
